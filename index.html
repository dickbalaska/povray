Dick Balaska's POV-Ray

This clone of POV-Ray contains two branches:

  • network - This branch contains povrayws, a version of POV-Ray that replaces the front end with websockets. This program is then slaved to my IDE qtpov. Tested on Linux and Windows.
  • qmake - This branch replaces the automake build process with qmake. Also it integrates the qtpov IDE for one big happy binary. Currently tested only on Linux.
