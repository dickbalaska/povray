Statusbar

December 13, 2017

The Statusbar shows information about the current running render.

There are three components:

  • Left - Shows progress on the current image. The number of pixels and percentage that have been rendered.
  • Middle - Optional. If you are rendering an animation, this shows the current frame and total number of frames to render.
  • Right - Timer that shows the elapsed time since the start of this job in HH:MM:SS format. If hours is 0, then it only shows MM:SS.
