qtpovray

October 18, 2017
qtpovray is an IDE for developing POV-Ray scenes.
It is a thin wrapper around POV-Ray and manages running POV-Ray and displaying rendering images.
Linux and Windows are supported.

The first time you run qtpovray, it figures out reasonable defaults and walks you through what it must.
